API 调用示例
多语言调用验证码识别API，极速接入，安全可靠
接口地址 /ocr，API Key 可在"个人中心"获取
请将 <你的API_KEY> 替换为你的专属密钥
💻curl 示例 
curl -X POST \
  -H "x-api-key: <你的API_KEY>" \
  -F "file=@captcha.png" \
  http://127.0.0.1:8000/ocr
🐍Python requests 示例 
import requests
headers = {"x-api-key": "<你的API_KEY>"}
files = {"file": open("captcha.png", "rb")}
r = requests.post("http://127.0.0.1:8000/ocr", headers=headers, files=files)
print(r.json())
🌐JavaScript fetch 示例 
const formData = new FormData();
formData.append('file', fileInput.files[0]);
fetch('http://127.0.0.1:8000/ocr', {
  method: 'POST',
  headers: { 'x-api-key': '<你的API_KEY>' },
  body: formData
})
.then(res => res.json())
.then(data => console.log(data));
